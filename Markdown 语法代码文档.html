19. 脚注 (Footnotes)

Typora 现状： 必须要成对出现。你写完代码后，Typora 会自动把解释的内容搬运到文档最底部。

  • 准确代码：

    Markdown

    xxxxxxxxxx
    Markdown是一种轻量级语言[^1]。
    

    [^1]: 这里是脚注的解释内容。

  • 案例效果： 文中会出现一个小小的数字 [1]，鼠标悬停会显示解释，点击会跳到底部。

    Markdown是一种轻量级语言1。

20. 锚点跳转 (Internal Links)

Typora 现状： 跳转的目标必须是标题。Typora 的规则是：把标题里的空格变成连字符，英文全部转小写。

  • 准确代码： 假设你有一个标题叫 ## My Project：

    Markdown

    xxxxxxxxxx
    [点击跳转到项目介绍](#my-project)

    假设你有一个中文标题叫 ## 总结：

    Markdown

    xxxxxxxxxx
    [点击跳转到总结](#总结)

  • 注意： 括号里必须是 # 加上标题的文本。

    Markdown

    点击跳转到总结

    在 Typora 的编辑模式下，直接用鼠标左键点链接是无效的（因为编辑器以为你想把光标移过去改字）。

    解决方法： 按住键盘上的 Ctrl 键（Mac 是 Command），同时点击那个链接。

21. 转义字符 (Escape)

Typora 现状： 也就是让 Markdown 符号“失效”，显示符号本身。

  • 准确代码：

    Markdown

    xxxxxxxxxx
    \# 这不是标题
    \**这不是加粗\**

  • 案例效果： 屏幕上直接显示 # 这不是标题，而不是变大变黑。

# 这不是标题 *这不是加粗*

22. 居中与颜色 (HTML Style)

Typora 现状： Markdown 本身不支持这些，必须用 HTML。

  • 准确代码 (文字居中)：

    HTML

    xxxxxxxxxx
    <center>这段文字会居中显示</center>
这段文字会居中显示

  • 准确代码 (改颜色)：

    HTML

    xxxxxxxxxx
    <span style="color:red">这段文字是红色的</span>

  • 注意： 这种写法在导出 PDF 时依然有效。

这段文字是红色的

24. 上标/下标 (Sub/Sup) —— ⚠️ 需开启设置

Typora 现状： 默认是关闭的！ 如果你直接输代码没反应，请去 文件 -> 偏好设置 -> Markdown -> 勾选 上标 和 下标，然后重启 Typora。

  • 准确代码：

    Markdown

    xxxxxxxxxx
    H~2~O  (下标)
    X^2^   (上标)

  • 案例效果： 2 会变小跑到 H 的右下角；2 会变小跑到 X 的右上角。

H2O (下标) X2 (上标)

25. 高亮 (Highlight) —— ⚠️ 需开启设置

Typora 现状： 同样默认关闭。请去 偏好设置 里勾选 高亮，重启后生效。

  • 准确代码：

    Markdown

    xxxxxxxxxx
    ==这段文字背景会变黄==

  • 案例效果： 文字像被荧光笔涂过一样（通常是亮黄色背景）。

这段文字背景会变黄
